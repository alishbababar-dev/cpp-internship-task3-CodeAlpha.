# cpp-internship-task3-CodeAlpha.
A C++ Bank Management System developed using Object-Oriented Programming concepts. The system allows customer and account management, deposits, withdrawals, fund transfers, transaction history tracking, and account information display.
Bank Management System (C++)

This project is a simple Bank Management System developed in C++ as part of an internship task. It demonstrates the use of Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, and composition.

Features
Create and manage customers
Create and manage bank accounts
Deposit funds
Withdraw funds
Transfer funds between accounts
Store transaction history
View recent transactions
Display account details and current balance
Technologies Used
C++
Object-Oriented Programming (OOP)
STL Vector
Learning Outcomes
Class Design
Data Management
Transaction Processing
OOP Implementation in Real-World Applications
